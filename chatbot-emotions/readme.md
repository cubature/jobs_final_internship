chatbot-emotion
======================
https://github.com/romwaldtff/chatbot-emotions.git

We recommend every repository include a README, LICENSE, and .gitignore.



Recupérer le projet en local (1ère fois):
-----------------
1. Installer git
2. Ouvrir un bash ( git bash / cmd)
3. Se diriger dans le futur dossier parent du projet.
4. Cloner le projet : `git clone https://github.com/romwaldtff/chatbot-emotions.git`
5. Normalement une fenêtre de connexion à github s'ouvre : se connecter
6. Aller dans le dossier du porjet : `cd chatbot-emotions`


Recupérer les mises à jours du répo distant :
-----------------
1. Commit les fichier modifier afin de ne pas les perdre. cf: add/commit
2. Eecupérer les fichier distants : `git pull`
3. Si complication, crier la phrase suivante : help...

mettre à jours le répo distant :
-----------------
1. Visualiser nos modification en local : `git status`
2. Ajouter les fichiers à la prochaine version : `git add document.extention` ou `git add dossier`
3. Finaliser et Titrer la prochaine version : `git commit -m "Titre de la modification"`
4. Recupérer les mises à jours du répo distant. cf : Recupérer les mises à jours du répo distant.
